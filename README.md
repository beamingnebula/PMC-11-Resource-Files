# PMC 11 Resource Files

Official resource file repository for Parkview Medical College 11th Batch.

## Overview

This repository serves as the central storage and documentation hub for all academic resources, materials, and files referenced on the [Parkview Medical College 11th Batch official website](https://pmc11.vercel.app/). It provides organized access to course materials, notes, and other educational resources for batch members.

## Purpose

- **Centralized Storage**: All files from the official website are stored and version-controlled here
- **Easy Access**: Quick reference and download of academic materials
- **Documentation**: Proper cataloging and organization of resources by subject and type
- **Collaboration**: Enable batch members to contribute and update materials

## Repository Structure

```
/
├── PDF/               # Academic subject materials
│   ├── anatomy/
│   ├── physiology/
│   ├── biochemistry/
│   └── ...
├── notes/             # Lecture notes and study guides
├── past-papers/       # Previous exam questions and papers
├── presentations/     # Slides and presentation materials
└── misc/              # Other resources and documents
```

## File Naming Convention

Files should follow this naming pattern:
```
[SUBJECT]_[TYPE]_[TOPIC]_[DATE/VERSION].[extension]

Example: ANATOMY_NOTES_UpperLimb_2024-01.pdf
```

## Contributing

Batch members are encouraged to contribute resources:

1. Ensure files are properly named following the convention above
2. Place files in the appropriate directory
3. Update the file index if adding new materials
4. Verify files are clean and properly formatted before uploading
